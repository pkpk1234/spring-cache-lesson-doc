# 课程简介

课程目标：介绍如何在Spring中使用缓存

1. 什么是缓存
   1. 定义：缓存是透明地存储数据以便更快地响应后续请求的组件。缓存的数据可以是之前请求的结果，如数据库的query cache；也可以是冗余的数据，比如web服务器缓存的静态页面。
   2. 目的：用空间换时间。
   3. 合理缓存的效果：快速、高效、可扩展。
2. 缓存的类型
   1. 本地缓存
   2. 分布式缓存
3. 缓存哪些数据
   1. 修改很少、直接读取代价较高的数据
